## 1. Halifax Regional Municipality (HRM)

HRM has **21 community plan areas**, each with its own Secondary MPS and Land Use By-law.
The zone code prefix usually identifies which LUB applies (e.g. HR- = Regional Centre,
ER- = some suburban plan areas, R- = various areas). When uncertain, look up the PID on
ExploreHRM first: https://www.halifax.ca/home/online-services/explorehrm

**HRM LUB Index page:** https://www.halifax.ca/city-hall/legislation-by-laws/land-use-by-laws
**HRM Community Plan Areas hub:** https://www.halifax.ca/about-halifax/regional-community-planning/community-plan-areas

### 1.1 Regional Centre ⭐ BUNDLED
- **LUB Name:** Regional Centre Land Use By-law
- **MPS Name:** Regional Centre Secondary Municipal Planning Strategy
- **Consolidated to:** Check halifax.ca for current consolidation date (frequently amended)
- **Official PDF URL:** https://www.halifax.ca/about-halifax/regional-community-planning/community-plan-areas/regional-centre-plan-area
  *(PDF links are on this page — fetch the page first to get the current PDF direct link)*
- **Zone prefixes:** HR-1, HR-2, CEN, COR, MU-1 through MU-4, CN, ER-1 through ER-4, I-1, I-2, P, OS
- **Bundled file:** `references/hrm-regional-centre-notes.md`
- **Notes:** Most frequently used LUB at Helio. Very large document (500+ pages). Contains Schmidtville Conservation District as an overlay. Subject to frequent amendment — always check consolidation date.

### 1.2 Halifax Plan Area (Halifax Mainland)
